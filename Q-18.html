Understanding Loops and Switch Cases in JavaScript

Loops in JavaScript

JavaScript provides various types of loops to execute a block of code multiple times efficiently. A loop continues to run as long as its condition remains true.

For instance, if you need to display “Hello World” five times, you can use a loop to achieve this with minimal code. The loop structure ensures that the code is executed repeatedly without redundancy: 

            <script>
            

            for (let i = 0; i < 5; i++) { 

            console.log("Hello World!"); } 

            </script>

Output:

  • Hello World!
  • Hello World!
  • Hello World!
  • Hello World!
  • Hello World!

Different Types of Loops

  • for Loop
  • while Loop
  • do-while Loop
  • for-in Loop
  • for-of Loop
  • Labeled Statement
  • Break Statement
  • Continue Statement
  • Infinite Loop (Error)

Switch Statement in JavaScript

The switch statement offers a streamlined way to execute different code blocks based on varying conditions, serving as an alternative to multiple if...else if...else statements.

Syntax:

            switch (expression) { 

            case value1: 

            // code block 1; 

            break; 

            case value2: 

            // code block 2; 

            break; 

            ... 

            default: 

            // default code block; 

            }
  • The expression is the value being compared.
  • case value1, case value2, etc., are possible matches for the expression.
  • The break statement exits the switch statement, preventing the execution from falling through to subsequent cases.
  • The default case is optional and runs if no cases match the expression.

How the Switch Statement Functions

  • Evaluation: The switch expression is evaluated once.
  • Comparison: The evaluated expression is compared with each case label using strict equality (===).
  • Execution: When a match is found, the corresponding code block is executed. If no match is found, the default case (if present) or the code after the switch block executes.
  • Break Statement: The break statement terminates the switch case, stopping the execution from proceeding to the next case unless the break is omitted (fall-through).
  • Default Case: The default case runs if none of the other cases match the expression.
