Create a Multi-Level Multiplication Function

This function allows us to multiply three numbers using a series of nested function calls. Each function takes one argument, and the result is obtained by multiplying all three values together.

Example of Use

The first function accepts the number X, the second one takes Y, and the final one receives Z. The result of the function will be X * Y * Z.

Function Implementation

function multiply(X) {
  return function(Y) {
    return function(Z) {
      return X * Y * Z;
    };
  };
}

Expected Output

console.log(multiply(2)(4)(5)); // Output: 40
console.log(multiply(12)(7)(11)); // Output: 924
