What are the different ways to declare a variable in JavaScript?

Answer:

In JavaScript, you can declare variables using var, let, or const.

  1. let – Introduced in modern JavaScript for block-scoped variables.
  2. var – An older way to declare variables. It's rarely used now, but understanding the difference between let and var is still important.
  3. const – Similar to let, but the value assigned to the variable cannot be changed after initialization.

Example Syntax:

var fullName = 'John Doe';

let occupation = 'Developer';

const country = 'USA';
